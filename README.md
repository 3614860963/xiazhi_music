# xiazhi_music

## 项目介绍

这是一个使用现代前端技术栈构建的音乐播放器项目，提供直观、美观的音乐播放体验。

### 核心功能
- 🎵 音乐播放控制（播放、暂停、上一曲、下一曲）
- 📱 响应式设计，适配不同设备屏幕
- 🎨 现代化UI界面，使用shadcn-ui和Tailwind CSS
- 🔄 自动播放列表管理
- 📖 音乐详情展示

### 技术特点
- 使用React和TypeScript确保类型安全和代码质量
- 基于Vite构建，提供快速的开发体验
- 集成shadcn-ui组件库，实现一致的设计风格
- 使用Tailwind CSS实现灵活的样式管理

## 如何编辑这段代码？

编辑应用有多种方式：

**使用 Vesa**

在 Vesa 中开始提示以进行更改。

通过 Vesa 所做的更改将自动提交到该仓库。

**使用你喜欢的 IDE**

如果你想用自己的 IDE 本地工作，可以克隆这个仓库并推送更改。推送的变更也会反映在 Vesa 中。

唯一的要求是安装了 Node.js 和 NPM——[用 NVM 安装](https://github.com/nvm-sh/nvm#installing-and-updating)

请按照以下步骤操作：

```sh
# 步骤 1：使用项目的 Git URL 克隆仓库。
git clone <YOUR_GIT_URL>

# 步骤 2：导航到项目目录。
cd <YOUR_PROJECT_NAME>

# 步骤 3：安装必要的依赖。
npm i

# 步骤 4：启动带有自动重载和即时预览的开发服务器。
npm run dev
```

**直接在 GitHub 编辑文件**

- 导航到想要的文件。
- 点击文件视图右上角的"编辑"按钮（铅笔图标）。
- 做出你的修改并提交这些变化。

**使用 GitHub Codespaces**

- 导航到你仓库的主页。
- 点击右上角附近的"代码"按钮（绿色按钮）。
- 选择"代码空间"标签。
- 点击"新代码空间"以启动新的代码空间环境。
- 直接在代码空间内编辑文件，完成后提交和推送你的更改。

## 我该如何部署这个项目？

### 在阿里云函数计算中部署

1. **登录阿里云控制台**：访问 [阿里云控制台](https://console.aliyun.com/) 并登录您的账号。
2. **创建函数计算服务**：
   - 导航到函数计算服务
   - 点击"创建函数"
   - 选择"HTTP 触发器"模板
   - 配置函数名称、运行环境（选择 Node.js 16.x 或更高版本）
3. **部署代码**：
   - 构建项目：在本地运行 `npm run build` 生成 `dist` 目录
   - 将 `dist` 目录内容上传到函数计算
   - 配置函数入口和处理方式
4. **配置触发器**：设置 HTTP 触发器，获取访问 URL

### 在阿里云 Pages 中部署

1. **登录阿里云控制台**：访问 [阿里云控制台](https://console.aliyun.com/) 并登录您的账号。
2. **创建 Pages 项目**：
   - 导航到 Pages 服务
   - 点击"创建项目"
   - 选择"前端构建"模式
3. **连接代码源**：
   - 选择 GitHub 作为代码源
   - 授权阿里云 Pages 访问您的 GitHub 仓库
   - 选择 `xiazhi_music` 仓库
4. **配置构建**：
   - 构建命令：`npm run build`
   - 输出目录：`dist`
   - 选择 Node.js 版本：16.x 或更高
5. **部署**：点击"部署"按钮，等待部署完成

## 我可以把自定义域名连接到我的项目吗？

可以的！

### 阿里云 Pages 自定义域名配置
1. 在 Pages 项目管理页面，点击"域名管理"
2. 点击"添加域名"
3. 输入您的自定义域名
4. 按照提示在域名服务商处配置 DNS 解析
5. 等待域名验证通过后，您的项目即可通过自定义域名访问

### 阿里云函数计算自定义域名配置
1. 在函数计算服务页面，点击"触发器管理"
2. 选择您的 HTTP 触发器
3. 点击"绑定自定义域名"
4. 按照提示完成域名绑定和 SSL 证书配置
